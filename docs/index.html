ZevFocus CLI Documentation

A blazing-fast CLI to track deep work, build focus streaks, and share tamper-proof progress.

📦 Installation

Windows:

irm https://2003aditya.github.io/ZevFocus-website/install.ps1 | iex

Linux (x86_64):

curl -sSL https://2003aditya.github.io/ZevFocus-website/install.sh | bash

This installs zevfocus globally in your system.

🚀 Quick‑Start

                    $ zevfocus login YourName                     # 🔐 Login to ZevFocus
                    $ zevfocus application                        # 🕵️ Detect current active application
                    $ zevfocus whitelist --application < >        # ✅ Add apps to whitelist
                    $ zevfocus whitelist --list                   # 📋 View whitelist
                    

                    $ zevfocus start --taskname "Write blog post" # ▶️ Start a focused session
                    # ...stay focused...
                    $ zevfocus stop                               # ⏹️ Stop the current session

                    $ zevfocus summary                            # 📈 View session summaries
                    $ zevfocus share                              # 📤 Generate a tweet + hash to share
                    $ zevfocus logout                             # 🚪 Logout from ZevFocus
                    $ zevfocus version                            # 🔢 Check ZevFocus version

🔐 Login

Register your username (stored locally and HMAC verified):

zevfocus login YourName

🕵️ Application Tracking

Test detection of currently focused application:

zevfocus application

📋 Manage Whitelist

Only whitelisted apps/tabs are tracked:

zevfocus whitelist --application "code,sublime"
zevfocus whitelist --list

🎯 Start a Focus Session

zevfocus start --taskname "Design new UI"

Runs in background and tracks time spent productively.

🛑 Stop Session

zevfocus stop

Ends your session and prints summary:

🎯 Session Ended.
                    🕒 Focus Duration: 110 minutes
                    🔥 Streak: 3 days

📊 Summary

View your overall productivity data:

zevfocus summary

                    🔥 Streak: 4 days

🧠 How It Works

  • Local data stored at ~/.zevfocus/
  • Session hashes use HMAC for tamper-proof records
  • Idle detection filters out non-productive time
  • Nothing is sent to a server unless you choose to share

🛠 Troubleshooting

  • Command not found? Make sure it's in your $PATH
  • Idle not detected? You may not have used mouse/keyboard recently
  • Want to reset? Delete ~/.ZevFocus

❓ FAQ

  • Is it offline? Yes, fully local by default
  • Is my data stored remotely? No, unless you share manually
  • Can I script it? Absolutely — works in shell, cron, or CI
