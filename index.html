ZevFocus

Hack your time, not just your code — a blazing‑fast CLI to track deep‑work sessions, build 🔥 streaks, and share tamper‑proof wins.

✨ Features at a Glance

  • ⚡️ Instant Sessions
    Start and stop with zevfocus start/stop.
  • 🛡 Tamper‑Proof
    HMAC signatures + hash to prove authenticity.
  • 🔥 Daily Streaks
    Dopamine‑driven streak counter like GitHub.
  • ⏱ Idle Detection
    Counts only real keyboard/mouse activity.
  • 📊 Beautiful Summaries
    CLI dashboards with emojis & structure.
  • 🐦 One‑Click Sharing
    Generate tweet‑ready cards + hash IDs.

🚀 Quick‑Start

                $ zevfocus login YourName                     # 🔐 Login to ZevFocus
                $ zevfocus application                        # 🕵️ Detect current active application
                $ zevfocus whitelist --application < >        # ✅ Add apps to whitelist
                $ zevfocus whitelist --list                   # 📋 View whitelist
                

                $ zevfocus start --taskname "Write blog post" # ▶️ Start a focused session
                # ...stay focused...
                $ zevfocus stop                               # ⏹️ Stop the current session

                $ zevfocus summary                            # 📈 View session summaries
                $ zevfocus share                              # 📤 Generate a tweet + hash to share
                $ zevfocus logout                             # 🚪 Logout from ZevFocus
                $ zevfocus version                            # 🔢 Check ZevFocus version
